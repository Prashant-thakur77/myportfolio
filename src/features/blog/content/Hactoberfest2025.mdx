## Organizations & Work

### Learning Equality

A major portion of my Hacktoberfest contributions focused on **Learning Equality**, an organization dedicated to providing equitable access to education worldwide.

My work included:

- Migrating UI components from Vuetify to the Kolibri Design System (KDS)
- Improving accessibility and visual consistency
- Refactoring components for better reusability and clarity
- Aligning code with design system standards
- Iterating on feedback through maintainer reviews

This required understanding a large, evolving codebase and adhering to strict contribution guidelines.
